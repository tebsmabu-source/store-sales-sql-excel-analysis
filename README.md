🟨 Project Overview

This project analyzes retail store performance using SQL and Excel.
The goal is to identify high-performing stores, customer patterns, and sales efficiency.

🟩 Tools Used
MySQL (SQL Workbench)
Microsoft Excel
GitHub
🟧 Dataset Description
Column	Description
Store_ID	Unique store ID
Store_Area	Size of store
Items_Available	Stock available
Daily_Customer_Count	Daily customers
Store_Sales	Total sales
Performance	High/Medium/Low category
🟦 SQL Analysis Performed
Created database and table
Inserted dataset
Retrieved and analyzed data
Calculated key business metrics
📌 Key SQL Insights
Highest performing store identified
Average sales calculated
Total customer count analyzed
Sales efficiency (sales per customer) computed
Performance categories grouped
📊 Key Results
Stores with higher customer count generate higher sales
Sales efficiency helps identify best-performing stores
Most stores fall under “High” performance category
